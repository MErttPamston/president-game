## 🎮 О игре

**Президент: Путь в будущее** — браузерная стратегия с пиксельной эстетикой, в которой вы принимаете судьбоносные решения во главе государства. Начиная с 1200 года, вы ведёте страну сквозь Средние века, Возрождение, Индустриализацию и Атомный век — вплоть до 2300 года и выхода в космос.

Каждый ход — новое испытание. Засухи, войны, торговые кризисы, народные восстания, дворцовые интриги, мятежи армии. Четыре ресурса, восемь видов угроз, десятки уникальных концовок. Один неверный шаг — и государство рухнет.

**Игра запускается прямо из браузера. Никаких зависимостей, никаких установок — один HTML-файл.**

## ✨ Возможности

### 🏛️ Глубокий геймплей

- **Четыре взаимозависимых ресурса** — Природа, Народ, Армия, Казна. Каждый влияет на остальные: мало золота — армия дезертирует, мало народа — налоги иссякают, мало природы — народ болеет.
- **Система стабильности** — интегральный показатель здоровья государства. Чем ниже — тем чаще и тяжелее кризисы.
- **Пассивный распад** — ресурсы медленно убывают каждый ход, даже если не делать ничего. Армия стоит денег на содержание. Природа деградирует. Игнорировать государство нельзя.
- **Последствия решений** — выбор не заканчивается на текущем ходу. Деferred-эффекты (рябь) возвращаются через несколько ходов в виде новых событий, вытекающих из прошлых решений.

### ⚠️ Система угроз

Новая панель **УГРОЗЫ ГОСУДАРСТВУ** отображает активные опасности с таймером обратного отсчёта:

| Угроза | Условие появления | Эффект при срабатывании |
|--------|------------------|------------------------|
| ☠️ Голодный бунт | Народ < 35 тыс. | −12к народа, −4к казны |
| 💀 Мятеж армии | Мало золота при большой армии | −15к армии, −6к казны |
| 🔥 Экологический коллапс | Природа < 25% | −12% природы, −4к народа |
| 💸 Казна на нуле | Золото < 15 тыс. | −10к казны, −5к армии, −3к народа |
| ⚔️ Вторжение | Слабая армия + низкая стабильность | −12к армии, −6к народа |
| 🦠 Эпидемия | Мало народа и золота | −14к народа, −3% природы |
| 🌑 Анархия | Стабильность < 20% | −8 по всем ресурсам сразу |
| 🌊 Стихийное бедствие | Природа < 20% | −10% природы, −5к народа, −6к казны |

Угрозу можно погасить, улучшив соответствующий ресурс. Если этого не сделать — она **срабатывает**, нанося тяжёлый удар.

### 📅 Эпохи и прогресс

Игра охватывает свыше 1000 лет истории через девять эпох:

```
СРЕДНИЕ ВЕКА → ВОЗРОЖДЕНИЕ → НОВОЕ ВРЕМЯ → ПРОСВЕЩЕНИЕ
→ ИНДУСТРИАЛИЗАЦИЯ → АТОМНЫЙ ВЕК → ИНФОРМАЦИОННЫЙ ВЕК
→ ПОСТИНДУСТРИАЛЬНЫЙ → КОСМИЧЕСКИЙ ВЕК
```

При смене эпохи происходит скачок лет, ресурсы меняются, создаётся автосохранение. Чем ближе к 2300 году — тем быстрее течёт время.

### 🎲 Разнообразие событий

Более **300 уникальных событий** четырёх типов:

- **Обычные события** — повседневные вызовы с двумя вариантами ответа. Выбор не очевиден: оба несут риски.
- **Квесты** — сюжетные задания с тремя вариантами и прописанными эффектами (голод в провинции, дракон на горе, чумной торговец, набег варваров).
- **Кризисы** — редкие, но катастрофические события. Объявление войны, народное восстание, чёрная смерть, пустая казна. Требуют немедленного решения.
- **События соседей** — дипломатические происшествия: подарки, набеги, блокады, смена правителей.

### 🏆 Несколько концовок

Финал зависит от того, что вы развивали на протяжении игры:

| Концовка | Условие |
|----------|---------|
| 👑 Золотой век | Слава ≥ 200 и среднее ресурсов ≥ 70% |
| 🌿 Зелёная утопия | Природа ≥ 80% |
| ⚔️ Военная империя | Армия ≥ 80 тыс. |
| 💰 Торговая держава | Казна ≥ 80 тыс. |
| 🏛️ Народная республика | Народ ≥ 80 тыс. |
| 🚀 В будущее! | Выжить до 2300 года |
| 📜 Выжили | Добраться до конца любой ценой |

### 🎵 Ретро-музыка и звук

Встроенный синтезатор воспроизводит четыре культовых трека через Web Audio API — без внешних файлов:

- 🍄 **Марио** — бодрая тема для спокойных времён
- 🧩 **Тетрис** (Коробейники) — задумчивое управление
- 🗡️ **Зельда** — эпические решения
- 🌑 **Имперский марш** — тёмная эпоха

### 💾 Сохранения

- **Локальное сохранение** — в localStorage браузера, сохраняется между сессиями
- **Код сохранения** — зашифрованная строка, которую можно скопировать и передать. Позволяет продолжить игру на другом устройстве.

### 🎨 Пиксельная графика

Каждое событие сопровождается уникальным пиксельным спрайтом, нарисованным на Canvas: замки, солдаты, торговцы, драконы, корабли, черепа, вулканы, волшебники и многое другое.

## 🕹️ Как играть

### Начало игры

При загрузке вы начинаете в **1200 году** с начальными ресурсами:

```
🌿 Природа:  50%       👥 Народ:  40 тыс.
⚔️  Армия:   20 тыс.   💰 Казна:  40 тыс.
```

Это скромный старт — управляйте ресурсами бережно, особенно в начале.

### Интерфейс

```
┌──────────────────────────────────────────┐
│   ГОД · ХОДЫ · ДНЕЙ · СЛАВА             │ ← Статистика
│   ⚖ СТАБИЛЬНОСТЬ [████████░░░] 80%      │ ← Здоровье государства
│   🌿 ПРИРОДА  ████████░░ 76%            │
│   👥 НАРОД    ██████░░░░ 58 тыс.        │ ← Ресурсы
│   ⚔️ АРМИЯ    ████░░░░░░ 38 тыс.        │
│   💰 КАЗНА    ██████████ 92 тыс.        │
├────────────────────┬─────────────────────┤
│                    │ ХРОНИКИ             │
│  📋 КАРТОЧКА       │ (лог событий)       │
│  СОБЫТИЯ           ├─────────────────────┤
│                    │ СОХРАНЕНИЕ          │
│  [ВАРИАНТ А]       ├─────────────────────┤
│  [ВАРИАНТ Б]       │ ⚠️ УГРОЗЫ           │
│                    │ (активные угрозы    │
│  ▶▶ СЛЕДУЮЩИЙ ХОД │  с таймерами)       │
└────────────────────┴─────────────────────┘
```

### Основные действия

**1. Читайте карточку события** — в ней описана ситуация и два-три варианта ответа. Каждый вариант меняет ресурсы по-своему, причём эффекты не всегда очевидны.

**2. Принимайте решение** — нажмите на один из вариантов. После этого появится карточка последствий с описанием того, что произошло, и изменениями ресурсов.

**3. Нажмите «Следующий ход»** — начнётся следующий день правления, ресурсы пройдут пассивный распад, могут сработать угрозы.

**4. Следите за угрозами** — если в панели «Угрозы» появился счётчик, у вас есть N ходов, чтобы поднять соответствующий ресурс выше порогового значения. Иначе угроза сработает.

### Советы по выживанию

- **Не игнорируйте Казну** — она нужна постоянно. Армия дезертирует без жалованья, события требуют золота. Держите выше 25 тыс.
- **Природа падает всегда** — она медленно деградирует каждый ход. Регулярно принимайте решения в её пользу, иначе экологический коллапс неизбежен.
- **Стабильность — главный индикатор** — если она падает ниже 40%, кризисы начнут сыпаться один за другим. Не допускайте провала любого ресурса ниже 20.
- **Смотрите на таймеры угроз** — угроза с таймером «2 хода» — это срочно. Откажитесь от выгодного события ради спасения ресурса.
- **Помните о рябях** — некоторые решения возвращаются к вам через 3–8 ходов. Жадное решение сейчас может стоить дорого позже.

### Условия поражения

Игра заканчивается поражением, если **любой ресурс упадёт до нуля**:

| Ресурс | Причина поражения |
|--------|------------------|
| 🌿 Природа = 0% | Экологическая катастрофа — земля мертва |
| 👥 Народ = 0 | Вымирание — некому управлять |
| ⚔️ Армия = 0 | Завоёваны — враги захватили страну |
| 💰 Казна = 0 | Банкротство — государство рухнуло |

## 🚀 Запуск

Никаких зависимостей. Просто скачайте файл и откройте в браузере:

```bash
git clone https://github.com/MErttPamston/president-game
cd president-game
# Откройте index.html в любом браузере
```

Или откройте напрямую через `Файл → Открыть` в Chrome, Firefox, Safari, Edge.

**Требования:** любой современный браузер с поддержкой Web Audio API (Chrome 66+, Firefox 60+, Safari 14+).
